# FLYVPN Packaged Application

This directory contains the "unpacked" version of the FLYVPN application for Windows.

## Contents

- `flyvpn.bat`: The executable launcher script. Double-click this to start the application.
- `index.html`: The main user interface file for the application.
- `icon.svg`: The official application icon.

## How to Run

To start FLYVPN, simply double-click the `flyvpn.bat` file.

This will open the application in your default web browser. All application logic is self-contained and will run correctly.
